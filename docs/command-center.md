# Command Center

Use this sequence to run the team with any capable AI assistant.

## Step 1: Strategy

Give the assistant:

- your completed song brief
- the Strategy Lead prompt

Ask for a full strategy output.

## Step 2: Trend + Audio Analysis

Next, give:

- the strategy output
- the Trend Scout prompt
- the Audio Hook Architect prompt

Ask for 10 concepts and the best first audio clip to test.

## Step 3: Script Production

Then provide:

- the top 3 concepts
- the Script Writer prompt

Ask for:

- 3 artist-post versions
- 3 fan-post versions
- 3 micro-creator versions

## Step 4: Video Packaging

Then provide:

- the top concepts
- the Script Writer output
- the Visual Director prompt
- the Edit Agent prompt
- the Publishing Agent prompt

Ask for a complete video package for each concept:

- opening frame
- shot list
- assets needed
- timeline editing notes
- on-screen text
- caption
- first comment
- artist-led and faceless variants

## Step 4A: Corecore Research Mode

If the concept is Corecore-style, provide:

- the emotional angle
- the target song clip
- the Corecore Researcher prompt
- the Asset Sourcing Agent prompt

Ask for:

- weird, funny, emotional, or uncanny reference material
- royalty-free replacements and supporting final assets
- a completed asset log

## Step 5: Seeding

Provide:

- the best concepts
- the Creator Matchmaker prompt

Ask for:

- creator categories
- outreach messaging
- a first-wave seeding order

## Step 6: Review

After posting, provide:

- performance metrics
- comment patterns
- reuse count
- the Experiment Analyst prompt

Ask for what to scale, revise, and kill.

## Operating Rule

Do not ask the system to generate random ideas endlessly.

Always force the team to answer these questions:

- what exact behavior does this audio enable?
- why would another creator copy it?
- what is the easiest first version to test?
- what exact assets are needed to make the video today?
